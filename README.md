# go-mysql-crud
Sample crud operation using Golang and MySql

## API ENDPOINTS

### All Characters
- Path : `/characters`
- Method: `GET`
- Response: `200`

### Create Post
- Path : `/characters`
- Method: `POST`
- Response: `201`

### Details a Post
- Path : `/characters/{slug}`
- Method: `GET`
- Response: `200`

### Update Post
- Path : `/characters/{slug}`
- Method: `PUT`
- Response: `200`

### Delete Post
- Path : `/characters/{slug}`
- Method: `DELETE`
- Response: `204`
